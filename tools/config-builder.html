Papfast Config Builder

这是给私有仓库使用的可视化配置页。你可以在这里填写真实 SMTP、LLM、多模块订阅条件，然后直接生成 config.local.json 和 GitHub Secret 所需的 Base64 文本。

怎么用

  1. 先填写下面的全局配置：SMTP、发件邮箱、LLM。
  2. 再添加一个或多个模块，每个模块对应一组论文订阅主题。
  3. 点 Generate Config，检查右侧生成结果。
  4. 点 Copy Base64 Secret。
  5. 去私有仓库 GitHub 页面，进入 Settings → Secrets and variables → Actions → Secrets，新建 Secret：PAPFAST_CONFIG_JSON_BASE64，把刚才复制的内容粘进去。
  6. 再去 Settings → Secrets and variables → Actions → Variables，确认变量 PAPFAST_PERSIST_STATE=true。
  7. 最后在 GitHub Actions 里手动运行一次 Daily Paper Push。
